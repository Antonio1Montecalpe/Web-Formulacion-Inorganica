Saberes básicos de la formulación inorgánica (hidruros, óxidos, hidróxidos y sales binarias)

1. Hidruros

¿Qué son? Compuestos de hidrógeno con otro elemento. El hidrógeno actúa con valencia -1.

¿Cómo se formulan? Se escribe el elemento y luego el H. Se cruzan las valencias.

Ejemplo: Calcio (+2) con H (-1) → CaH₂

¿Cómo se nombran?

  • Stock: Hidruro de + nombre del elemento + (valencia)
  • Tradicional: -oso / -ico

2. Óxidos

¿Qué son? Compuestos de oxígeno con otro elemento. El oxígeno actúa con valencia -2.

¿Cómo se formulan? Se escribe primero el elemento y luego el oxígeno. Se cruzan las valencias.

Ejemplo: Fe (+3) con O (-2) → Fe₂O₃

¿Cómo se nombran?

  • Sistemática: dióxido de carbono
  • Stock: óxido de hierro (II)
  • Tradicional: óxido férrico

3. Hidróxidos

¿Qué son? Compuestos de un metal con el grupo OH⁻.

¿Cómo se formulan? Se escribe el metal seguido de (OH), cruzando valencias.

Ejemplo: Al (+3) con OH (-1) → Al(OH)₃

¿Cómo se nombran?

  • Stock: hidróxido de hierro (III)
  • Tradicional: hidróxido ferroso

4. Sales binarias

¿Qué son? Compuestos de un metal y un no metal, sin oxígeno.

¿Cómo se formulan? Se escribe primero el metal y luego el no metal. Se cruzan valencias.

Ejemplo: Fe (+3) con Cl (-1) → FeCl₃

¿Cómo se nombran?

  • Stock: Cloruro de hierro (III)
  • Tradicional: Cloruro férrico
