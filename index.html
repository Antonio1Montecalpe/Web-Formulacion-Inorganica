Saberes básicos de la formulación inorgánica (hidruros, óxidos, hidróxidos y sales binarias)

1. Hidruros

¿Qué son? Compuestos de hidrógeno con otro elemento. El hidrógeno actúa con valencia -1.

¿Cómo se formulan? Se escribe el elemento y luego el H. Se cruzan las valencias.
Ejemplo: Calcio (+2) con H (-1) → CaH₂

¿Cómo se nombran?
Stock: Hidruro de + nombre del elemento + (valencia). Ej: NaH → hidruro de sodio
Tradicional: sufijos -oso / -ico. Ej: FeH₂ → hidruro ferroso




2. Óxidos

¿Qué son? Compuestos formados por oxígeno y otro elemento. El oxígeno actúa con valencia -2.

¿Cómo se formulan? Se escribe el elemento seguido del oxígeno. Se cruzan las valencias. Ej: Fe₂O₃

¿Cómo se nombran?
Sistemática: dióxido de carbono
Stock: óxido de hierro (II)
Tradicional: óxido ferroso, óxido férrico




3. Hidróxidos

¿Qué son? Compuestos de un metal con el grupo OH⁻ (hidroxilo).

¿Cómo se formulan? Metal seguido del grupo OH. Se cruzan las valencias. Ej: Al(OH)₃

¿Cómo se nombran?
Stock: hidróxido de hierro (III)
Tradicional: hidróxido férrico




4. Sales Binarias

¿Qué son? Compuestos de un metal y un no metal, sin oxígeno.

¿Cómo se formulan? Metal seguido del no metal. Se cruzan valencias. Ej: NaCl

¿Cómo se nombran?
Stock: cloruro de sodio
Tradicional: cloruro férrico
