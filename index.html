Saberes básicos de la formulación inorgánica

(Hidruros, Óxidos, Hidróxidos y Sales Binarias)

1. Hidruros

¿Qué son? Compuestos de hidrógeno con otro elemento. El hidrógeno actúa con valencia -1.

¿Cómo se formulan? Se escribe el elemento y luego el H. Se cruzan las valencias.

Ejemplo: Calcio (+2) con H (-1) → CaH₂

¿Cómo se nombran?

  • Stock: Hidruro de + nombre del elemento + (valencia)
  • Tradicional: Sufijos -oso / -ico

Ejercicio 2: ¿Cómo se llama FeH₃ en nomenclatura Stock?

Ejercicio 3: ¿Cómo se nombra FeH₂ en nomenclatura tradicional?

2. Óxidos

¿Qué son? Compuestos con oxígeno y otro elemento (valencia -2).

¿Cómo se formulan? Elemento + oxígeno, cruzando valencias. Ej: Fe(+3) + O(-2) → Fe₂O₃

¿Cómo se nombran?

  • Sistemática: dióxido, trióxido, etc.
  • Stock: óxido de + elemento (valencia)
  • Tradicional: óxido ferroso/férrico

Ejercicio 1: ¿Cómo se llama Fe₂O₃ en nomenclatura tradicional?

Ejercicio 2: ¿Cuál es la fórmula del óxido de hierro (II)?

Ejercicio 3: ¿Cómo se nombra CO₂ en sistemática?

3. Hidróxidos

¿Qué son? Compuestos de un metal con el grupo OH⁻.

¿Cómo se formulan? Metal + grupo OH, cruzando valencias. Ej: Al(OH)₃

¿Cómo se nombran?

  • Stock: hidróxido de + metal + (valencia)
  • Tradicional: hidróxido ferroso/férrico

Ejercicio 1: ¿Cuál es la fórmula del hidróxido de calcio?

Ejercicio 2: ¿Cómo se nombra Fe(OH)₂ en tradicional?

Ejercicio 3: ¿Cómo se nombra Al(OH)₃ en nomenclatura Stock?

4. Sales binarias

¿Qué son? Compuestos de un metal y un no metal (sin oxígeno).

¿Cómo se formulan? Metal + no metal, cruzando valencias. Ej: NaCl

¿Cómo se nombran?

  • Stock: nombre del no metal -uro + de + metal + (valencia)
  • Tradicional: cloruro ferroso/férrico

Ejercicio 1: ¿Cómo se nombra NaCl en Stock?

Ejercicio 2: ¿Cuál es la fórmula del cloruro ferroso?

Ejercicio 3: ¿Cómo se llama FeCl₃ en Stock?
